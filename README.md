# drdaveheath-tools

A Claude Code plugin marketplace by Dr Dave Heath.

## Install

```
/plugin marketplace add DrDaveHeath/drdaveheath-tools-993cf8
/plugin install transcript-to-blog
/reload-plugins
```

## Plugins

- **transcript-to-blog** — turn a video transcript into a publish-ready blog post in your voice, saved to Google Drive.

The plugin's own install and usage guide lives in [`transcript-to-blog/README.md`](transcript-to-blog/README.md).
