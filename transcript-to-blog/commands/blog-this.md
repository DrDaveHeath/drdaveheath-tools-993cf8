---
description: Turn a transcript into a blog post in your voice and save it to Google Drive
---

Run the transcript-to-blog skill.

If setup has not been done yet (no saved config, or Google Drive is not connected), run the first-run setup first.

Otherwise, take the transcript the user provides (pasted, named as a file, or "the latest" from the source folder), generate the blog by following the bundled voice files, and save it to the destination Google Drive folder in markdown and plain text. Then report the title and where it was saved.
